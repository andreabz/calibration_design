  • Show All Code

Author

The realistic complication

A clean constant-RSD model is useful for learning WLS, but real low-level analytical work often behaves differently. Near the LOQ, the response standard deviation may stop decreasing because blank noise, integration noise, or background correction dominates. Farther from the LOQ, relative error can become more stable.

This mixed behavior creates a problem for simple global weighting rules. A rule such as \(1/x^2\) assumes that variance continues to shrink rapidly as concentration approaches zero. If the method has a low-concentration noise floor, that assumption can give too much influence to the lowest calibrators.

  x = seq(1, 100, length.out = 300)

      method = "True mixed variance",

      method = "1/x",

The true weight curve flattens near the low end because precision does not keep improving indefinitely. The \(1/x^2\) curve keeps rising. That mismatch is exactly where routine low-level calibration can become fragile.

  n = 1000,
  xmin = 1,
  xmax = 100,
  mean_fun = mean_fun,
  variance_fun = sigma_fun
)

prediction_tables <- lapply(
  models,
  function(model_set) {
    lapply(
      model_set,
      predict_calibration_model,
      test_data = test_data,
      variance_fun = sigma_fun
    )
  }
)

Take-home message

WLS requires a variance model. A simple global rule may be convenient and may even be approximately right over part of the range, but that does not make it right near the LOQ. When the method has a low-concentration noise floor, the analyst should inspect low-level residuals, replicate behavior, and prediction performance separately from the global calibration summary.
